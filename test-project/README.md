# test-project

A Vela project created with template: web

## Getting Started

1. **Build the project:**
   ```bash
   vela build
   ```

2. **Run the project:**
   ```bash
   vela run
   ```

3. **Run tests:**
   ```bash
   vela test
   ```

## Project Structure

```
test-project/
├── src/           # Source code
├── tests/         # Unit tests
├── docs/          # Documentation
├── vela.toml      # Project configuration
└── README.md      # This file
```

## Development

- Format code: `vela fmt`
- Check syntax: `vela check`
- Build optimized: `vela build --opt-level aggressive`
