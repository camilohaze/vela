# Changelog

All notable changes to the Vela ORM will be documented in this file.

## [Unreleased]

### Added
- Initial release of Vela ORM
- Type-safe entity definitions with decorators
- Multiple database support (PostgreSQL, MySQL, SQLite)
- Connection pooling with health monitoring
- Type-safe query builder with fluent API
- Entity relations (one-to-one, one-to-many, many-to-many)
- Lazy and eager loading
- Migration system with automatic schema versioning
- Transaction management with savepoints and isolation levels
- Comprehensive error handling and validation
- Observability integration (metrics, tracing, logging)
- Comprehensive test suite
- Documentation and examples
